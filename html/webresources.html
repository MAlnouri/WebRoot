Web Resources

  • Stackoverflow
    Widely known as one of the best resources avaiable to developers, this site is a useful forum to find answers to problems you encounter and share your knowledge. Stackoverflow has over 16.5 million questions posted covering many fields including web, app or software development. Any developer should be able to find Stackoverflow useful with helping them find solutions to their bugs and issues, or if they are willing to help and share their solutions to other developers' problems. I will be using this site as a way to find help in understanding why I have certain problems in my apps or code, and use the help to develop my own solutions to the issue.
    Rating: 5 out of 5

  • CSS Reference
    This is a great resource for developing and styling your web pages using CSS, learning how each property works with visual guides. It has great visual and animated exmaples of CSS properties and how they work. It also lists all available sub properties. Anyone who is looking to create a web page or expand their knowledge and understanding of CSS properties will be able to find use of this website. I will personally use this resource as a reference to apply to my web pages. I will also use it to learn different CSS styling properties that I can use to make my web pages look more attractive and appealing.
    Rating: 4 out of 5

  • Codecademy
    This is an amazing resource for anyone trying to learn new programming languages in a variety of fields in computer science. Codecademy has a large amount of free tutorials and walkthroughs that teach you the basics of each language and you can create your own projects and learn how to develop interactive websites, games, and apps. Anyone who is looking to learn programming languages, or seeking a role in a variety of computer science positions should find the informations and tutorials Codecademy provides useful. I will be using this resource as a way to advance my knowledge in areas I have handled such as HTML and CSS, and use it to learn and expand my knowledge of areas such as Node and JS.
    Rating: 5 out of 5

  • Codewars
    Codewars is a site where you can go to learn and develop your programming skills by completing challenges and advancing your rank. Codewars has many challenges for a wide range of languages which you can use to learn or advance your knowledge of the language. You should find this site useful if you are looking to learn or advance your ability to solve problems in computer science related fields, or if you enjoy creative problem solving. I will be using this site to progress my understanding of the languages I will be using, as I believe it is important to be able to understand why your code does or does not work to be a developer.
    Rating: 4 out of 5

  • Visual Studio Code
    Visual studio code is a very useful, free code editor that provides support for a variety of programming languages. Visual studio code provides many useful extensions that support your coding needs such as a built in terminal, git commands, debugging and syntax highlighting. Any developer should be able to find visual studio code useful as it provides support for many different languages. I personally use this code editor as my primary code editor for developing my web apps and pages.
    Rating: 5 out of 5

  • W3Schools
    A great resource for developing good web pages and learning how to implement code in web development. It has coding syntax and examples for a wide variety of languages available including HTML, CSS, JS, Node, JSON, SQL and PHP. Anyone who is looking to create a web page or expand their knowledge of web development will find this resource useful. I will personally use this resource to look through different tutorials and examples of CSS style and JS code that I can use to make my web apps more modern and interactive.

    Rating: 5 out of 5
